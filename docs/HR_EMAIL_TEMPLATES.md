## Sign-off conventions

- Always sign as **HR Department / NB-Media** (NOT "NB Media Hiring Team" as the modal defaults use)
- Greeting uses **"Dear {{CandidateName}},"** (NOT "Hi {{firstName}},") for all candidate-facing emails
- Opening line is **"Greetings from NB Media!"** for nearly every candidate-facing template
- Closing is **"Warms Regards,"** (note: the doc uses "Warms" not "Warm" — preserved verbatim)
